Emojis e Unicodes

Unicodes

Para definir unicodes no html precisamos usar a tabela ASCII.

A tabela ASCII é uma correspondência entre linguagem de máquina e símbolos que utilizamos para se comunicar.

Exemplo

  • ! é (#33;)
  • & é (#38;)
  • 2 é (#50;)
  • * é (#42;)

Emojis

São caracters do conjunto de caracters unicodes do UTF-8.

Esses caracters são definidos no <meta charset="UTF-8">, porque antes do seu site ir para o browser é necessário ele reconhecer o conjunto de caracters da página

Exemplo

  • 😄 é (#128516;)
  • 😍 é (#128525;)
  • 💗 é (#128151;)

Atributos

Descrição

Os atributos html são elementos adicionais para definir seu comportamento e funcionalidade. Normalmente, eles são especificados dentro da tag de abertura < >, e são caracterizados com nome e valor, separado por um sinal de igual (=).

Exemplo

  • id
  • class
  • rel
  • type
  • style

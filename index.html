Objetivo

Fixar os conhecimento sobre tags da primeira aula do curso de html e css do tomorrow

Tags

  • <p>
  • <h1>
  • <br>
  • <hr>
  • <pre>

Descrição

    <p>

    Serve para colocar um parágrafo dentro do html.

    <h1>

    Serve para colocar um header/title no html. Ele tem uma hierarquia de tamaho o <h1> até <h6>

    <br>

    Serve para quebrar linha (break row).

    <hr>

    Quebra a linha colocando uma régua.

    <pre>

    É um parágrafo que preserva os espaços e quebras de linhas.
